## Descripción

Este proyecto es un **Generador de Códigos QR** totalmente responsive que funciona tanto en **PC como en móviles**, simulando la experiencia de una app nativa.  
Permite generar códigos QR a partir de texto, URLs, emails, números de teléfono, WiFi y más. Además, ofrece:

- **Modo claro y modo oscuro** para una mejor experiencia visual.  
- **Agregar un logo al QR** (por ejemplo, el logo de YouTube o cualquier imagen personalizada).  
- **Descargar el QR** como imagen PNG.  
- **Guardar QR en historial** usando `localStorage`.   


---

## Funcionalidades

1. **Generar QR:** A partir de texto, URL, email, teléfono o WiFi.  
2. **Descargar QR:** Guardar la imagen generada en PNG.  
3. **Guardar en historial:** Mantiene un historial local de los últimos 50 QR generados.  
4. **Modo claro y oscuro:** Cambia el tema según la preferencia del usuario.  
5. **Agregar logo:** Puedes personalizar el QR con tu logo favorito.  
6. **Responsive:** Se adapta perfectamente a móviles y PCs.  
7. **Ejemplos rápidos:** Botones predefinidos para generar QR instantáneamente.
